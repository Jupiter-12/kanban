# 变更：添加看板核心功能

## 原因

看板系统的核心价值是"让工作可见"，用户认证功能已完成，现在需要实现看板的核心能力，让用户能够创建项目、管理看板列和任务卡片。

## 变更内容

- 添加项目（Project）管理功能：创建、查看、编辑、删除项目
- 添加看板列（Column）管理功能：创建、查看、编辑、删除、排序列
- 添加任务卡片（Task）管理功能：创建、查看、编辑、删除任务
- 添加前端看板视图：展示项目列表、看板视图、任务卡片

## 影响

- 受影响的规格：新增 `kanban-core` 能力
- 受影响的代码：
  - 后端：新增 models、schemas、services、api 模块
  - 前端：新增 views、components、stores、api 模块
  - 数据库：新增 projects、columns、tasks 表
